h1. cilea

h2. Intro

This is the bot hanging out in #lanpartei on freenode.

h2. Running

# install "RVM":rvm.beginrescueend.com
# clone this repo, cd into it
# `gem install bundler --no-rdoc --no-ri`
# bundle
# guard

The last command is optional, it will start the bot in test mode. You can change the channel and name in the `config.yml` file.
